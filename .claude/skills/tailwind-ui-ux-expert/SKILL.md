# Tailwind UI/UX Expert

## Purpose
Helps the AI create highly accessible, responsive, and visually stunning UI components using Tailwind CSS v4 and Next.js 15, focusing on modern design patterns, micro-interactions, and premium aesthetics.

## Instructions
1. **Premium Aesthetics**: Use modern web design trends such as glassmorphism, subtle gradients, and tailored color palettes (e.g., `emerald` and `stone` as defined in the project conventions).
2. **Tailwind Best Practices**: Utilize Tailwind CSS v4 features efficiently. Avoid using arbitrary values (`w-[31px]`) when standard utility classes exist. Keep class strings clean and readable.
3. **Accessibility (a11y)**: Always use semantic HTML5 elements. Ensure all interactive components have appropriate `aria-labels`, roles, and proper keyboard focus management.
4. **Micro-Animations**: Add life to the interface by using subtle transitions on interactive elements (e.g., `transition-all duration-200 hover:scale-[1.02] active:scale-95`).
5. **Responsive Design**: Ensure mobile-first styling using breakpoints (`sm:`, `md:`, `lg:`) to guarantee the application works seamlessly on all device sizes.
6. **Dark Mode Ready**: Structure styles to easily adapt to dark mode (using `dark:` prefix) if enabled in the project.
