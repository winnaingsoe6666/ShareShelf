  <behavior>
    - Given an OAuth2User with sub="google123", email="alice@gmail.com", name="Alice", picture="https://..." — processOAuthUser finds existing user by googleId and returns it
    - Given an OAuth2User with email matching an existing LOCAL user — processOAuthUser links Google ID to that user, sets authProvider to GOOGLE, imports avatar if null, returns updated user
    - Given an OAuth2User with no matching googleId or email — processOAuthUser creates new User with name, email, empty passwordHash, authProvider=GOOGLE, googleId, avatarUrl from picture
    - generateOAuthResponse returns AuthResponse with token from JwtTokenProvider.generateToken and refresh token
  </behavior>
  <action>
    Create `OAuth2Service.kt` in `backend/src/main/kotlin/com/shareshelf/auth/` (per D-05, D-06, D-07, D-08):

    Class signature: `@Service class OAuth2Service(private val userRepository: UserRepository, private val jwtTokenProvider: JwtTokenProvider, private val refreshTokenRepository: RefreshTokenRepository, @Value("\${jwt.refresh-expiration-ms}") private val refreshExpirationMs: Long)`

    Method `processOAuthUser(oAuth2User: OAuth2User): User`:
    1. Extract `sub` (Google ID), `email`, `name`, `picture` from OAuth2User attributes
    2. Try `userRepository.findByGoogleId(googleId)` — return if found (per D-05)
    3. Try `userRepository.findByEmail(email)` — if found, link: set `googleId`, `authProvider = GOOGLE`, set `avatarUrl` if currently null and Google provides picture. Save and return (per D-05, D-07)
    4. Create new User: `name` from Google or email prefix, `email`, `passwordHash = ""`, `googleId`, `authProvider = GOOGLE`, `avatarUrl` from picture. Save and return (per D-07, D-08)

    Method `generateOAuthResponse(user: User): AuthResponse`:
    1. Generate JWT via `jwtTokenProvider.generateToken(user.id!!, user.email)`
    2. Generate refresh token following `AuthService.createRefreshToken()` pattern (hash with SHA-256, save RefreshToken entity, return raw token)
    3. Return `AuthResponse(token, refreshToken, userId, name, email, trustScore, community, avatarUrl)`

    Import: `org.springframework.security.oauth2.core.user.OAuth2User`, `com.shareshelf.auth.dto.AuthResponse`, `com.shareshelf.auth.entity.*`
